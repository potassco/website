_Clorm_ is a Python library that provides an Object-Relational Mapping
(ORM)-style interface to the clingo Answer Set Programming solver. It enables
users to assert facts into the solver and extract results in a straightforward
way. _Clorm_ is designed to complement the existing clingo API, making it
easier to develop and maintain Python applications that integrate with clingo.
